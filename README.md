DATA SCIENTIST WORKING PRE-TEST

Data Science project ini merupakan tugas yang saya kerjakan sebagai pre-test untuk menentukan saham terprofit selama 3 bulan terakhir, 1 tahun terakhir dan 5 tahun terakhir serta yng paling aman

Formula yang say gunakan:

Return = (Close_terakhir / Close_awal) - 1

Dimana:
Close_terakhir = Harga terakhir saham pada saat penutupan
Close_awal = Harga awal saham pada saat awal pembelian



PROJECT STRUCTURE
Project ini terdiri dari beberapa data:
- Data: terdiri dari dara .csv setiap Stcok yang diberikan
- notebooks: Jupyter notebooks untuk Persiapan, Pembersihan Data, Preprosesing Data, Perhitungan & Kesimpulan dan Visualisasi Data
- Menggunakan Pyhton 3.11.0
- README.md: Penjelasan tentang tugas dan lainnya.
